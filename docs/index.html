Cadmus Question Library Tools

Chrome extension for importing, editing, exporting, and managing question banks in Cadmus assessments.

Latest: loading...

Install (first time)

  1. Download the latest .zip from the Releases page:
    Download latest release
  2. Extract the ZIP to your preferred location
    Recommended: your user folder, e.g. ~/cadmus-plugin
  3. Open chrome://extensions in Chrome
  4. Enable Developer mode (toggle in the top-right corner)
  5. Click Load unpacked and select the extracted extension folder
  6. Pin the extension to your toolbar for easy access

Update (existing install)

  1. Download the latest .zip from the Releases page
  2. Extract the ZIP over your existing extension folder, replacing all files
  3. Open chrome://extensions
  4. Find Cadmus Question Library Tools and click the reload icon (circular arrow)
  5. Done — the extension is now running the latest version
The extension checks for updates automatically when you open the popup. If a newer version is available, you will see a blue banner at the top linking back to this page.
